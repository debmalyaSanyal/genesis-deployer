# 🚀 Genesis Deployer

A simple, AI-powered tool that takes a user's prompt, generates a complete web application, and creates a new GitHub repository for it.



## ✨ Features

-   **AI Code Generation:** Describe your application in plain English.
-   **GitHub Integration:** Automatically creates a new repository and commits the generated code.
-   **One-Click Deployment:** Creates a Vercel-ready project that can be deployed instantly.
-   **Secure Authentication:** Uses NextAuth.js for safe sign-in with GitHub.

## 💻 Tech Stack

-   **Framework:** Next.js (with App Router)
-   **Styling:** Tailwind CSS
-   **Authentication:** NextAuth.js
-   **AI Provider:** Groq (using `llama-3.1-8b-instant`)
-   **GitHub API:** Octokit.js
-   **Deployment:** Vercel
